# Open 3D Foundation (open-3d-foundation)
The Open 3D Foundation is a Linux Foundation project supporting the Open 3D Engine (O3DE), an open source, real-time 3D engine for building games, simulations, and other 3D applications. With Microsoft as a premier member, it provides a modular, cross-platform engine with advanced rendering capabilities.
